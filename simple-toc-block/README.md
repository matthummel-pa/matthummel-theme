# Simple TOC Block

A lightweight **Table of Contents** block for the WordPress block editor. Drop it
into a post and it automatically builds a clean, linked outline from your
headings — rendered server-side, so it's fast, SEO-friendly, and accessible.

This is the free core of a freemium product. See `CLAUDE.md` for the full
roadmap and the planned Pro features.

## Features (v0.1.0)
- Auto-generates the TOC from the post's H2/H3/H4 headings
- Toggle which heading levels to include
- Numbered or bulleted list
- Server-rendered (present in initial HTML) + accessible `<nav>` landmark
- Anchor IDs injected into headings automatically so links scroll correctly

## Develop
Requires Node.js 18+ and a local WordPress site.

```bash
npm install        # install build tooling (@wordpress/scripts)
npm run start      # watch + rebuild while developing
npm run build      # production build into /build
npm run plugin-zip # package an installable .zip
```

## Install (for testing)
1. Run `npm install && npm run build`.
2. Copy this folder into `wp-content/plugins/` of a local WordPress site
   (or zip it and upload via **Plugins → Add New → Upload**).
3. Activate **Simple TOC Block**, then add the **Table of Contents** block to a
   post that has some headings and preview it.

## Project layout
```
simple-toc-block.php   Plugin header + PHP helpers + heading-id filter
src/
  block.json           Block metadata, attributes, supports
  index.js             Block registration
  edit.js              Editor UI
  render.php           Server-side front-end render
  style.scss           Front-end styles
  editor.scss          Editor-only styles
build/                 Compiled output (generated by npm run build)
```
